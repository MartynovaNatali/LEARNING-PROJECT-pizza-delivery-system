🍕 Система заказа пиццы «Большой папа»

## 📌 О проекте

Проект представляет собой **спецификацию требований** к веб-сайту заказа пиццы для компании «Большой папа».  
Разработан в рамках курса по системному анализу и проектированию информационных систем.

**Цель проекта:** автоматизация процесса заказа пиццы с доставкой, сокращение времени ожидания до 45 минут, автоматическая выдача купонов при опоздании.

 📂 Структура

- `docs/` — документы (SRS, User Stories, RACI, стейкхолдеры)
- `images/` — прототипы и изображения, схемы и диаграммы (BPMN, UML, контекстная, ERD)
- `api/` — Postman коллекция
